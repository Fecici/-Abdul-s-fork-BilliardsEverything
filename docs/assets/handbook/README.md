# Handbook assets

This directory holds source-controlled visual evidence for
`docs/CODEBASE-HANDBOOK.md`.

Present source diagrams:

1. `architecture.mmd`: JavaFX -> task -> JNA -> native -> database/result ownership.
2. `autopolyvary-ordering.mmd`: coordinate/map/render/commit/candidate ordering.
3. `research-workflow.mmd`: vary, calculate, cover, triples, pattern, and manual loop.

Remaining captured/redrawn asset families:

1. current Abdul Windows main-view screenshot with numbered functional bands;
2. AutoPolyVary, Cover, and triples current-build screenshots;
3. triangle angle-coordinate and unfolding figures derived from the primary
   paper, redrawn with explicit source attribution rather than copied pages.

Screenshots must record commit, OS scaling, window size, database/fixture, and
whether the image is source-run or historical. Do not put an old jar screenshot
in the handbook as though it represented current Abdul source.
